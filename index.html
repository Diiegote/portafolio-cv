Food App

Este proyecto es una aplicación de página única (SPA) que consume información de una API Rest de recetas. Proporciona funcionalidades como filtrado, ordenamiento, búsqueda y creación de nuevas recetas. Las tecnologías utilizadas incluyen Node.js, PostgreSQL, React.js, Redux, HTML y CSS. Este proyecto individual me brindó la oportunidad de aplicar los conocimientos adquiridos durante mi formación, así como de desarrollar nuevas habilidades. Fue una experiencia en la que pude demostrar mis capacidades técnicas y mi capacidad para trabajar de manera autónoma.

Se desarrolló utilizando Node.js, PostgreSQL, React.js, Redux, HTML y CSS. Fue un proyecto individual donde pude demostrar mis habilidades y aplicar lo aprendido, logrando resultados profesionales y una experiencia interactiva para los usuarios.
